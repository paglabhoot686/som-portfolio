# Som Chakravarty — Portfolio

Personal portfolio site for [Som Chakravarty](https://www.linkedin.com/in/somchakravarty/) — Product Leader, UX Designer, AI Builder.

## Stack

- **Vite** + **React** — fast builds, modern DX
- **Tailwind CSS v4** — utility-first styling via `@tailwindcss/vite`
- **Vercel** — deploy-ready

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Deploy

Push to GitHub and connect to Vercel. Zero-config deployment — Vite is auto-detected.

Or deploy manually:

```bash
npx vercel
```

## Sections

1. **Hero** — Name, tagline, photo, CTAs
2. **About** — Career arc: UX → Product → AI
3. **What I Do** — Fractional leadership, app dev, UX audits
4. **Case Study** — Mission Control showcase
5. **Writing** — LinkedIn articles
6. **Contact** — Email + LinkedIn
7. **Footer** — Minimal

## Customization

- **Photo:** Replace the placeholder in the Hero section with an actual image
- **Email:** Update `somnath686@gmail.com` in Contact and Footer
- **Articles:** Update links in `Writing.jsx` with actual LinkedIn article URLs
- **Colors:** Modify theme tokens in `src/index.css` under `@theme`

## Design

Scandinavian minimal aesthetic. Typography-driven. Dark hero, light body. One accent color (`#c5705d` — a warm terracotta).
